# Индивидуальное задание — Вариант 17: Лифт Дилли

## Описание задачи

В экспериментальном лифте только две кнопки:
- **Кнопка 1** — подняться на один этаж вверх
- **Кнопка 2** — опуститься на один этаж вниз

Дилли катается на лифте, пока не побывает на каждом этаже хотя бы по одному разу. По данной последовательности нажатий кнопок определить **общее количество этажей в доме**.

## Примеры

| Входные данные | Ответ | Пояснение |
|---|---|---|
| `11` | 3 | Старт на 1-м: 1→2→3, посещены {1,2,3} |
| `21212` | 2 | Старт на 2-м: 2→1→2→1→2, посещены {1,2} |
| `1221221221221` | 6 | Старт на 5-м: посещены все 6 этажей |

## Ключевая идея алгоритма

Принять начальную позицию за 0. Каждое нажатие '1' → +1, '2' → -1.  
**Ответ = max\_pos − min\_pos + 1**
